## 🧠 The "Why" Behind the Design :

I wanted a site that felt like a glimpse into the future of the web. Here’s the philosophy I followed:

A "Glass" Feel: Using Glassmorphism, I created a UI that feels light and layered, with 20px blurs that give the site a sense of depth.

Motion as Feedback: From the "Magnetic" buttons that lean toward your cursor to the 3D-tilting project cards, every movement is designed to make the experience feel alive.

The Nebula Effect: A global 3D-inspired background remains fixed as you scroll, creating a consistent "space" for my work to live in.

Dark Mode by Choice: While the site defaults to a sleek Deep Onyx, I built a theme toggle because I believe a great developer always gives the user control over their environment.


## 🛠 My Toolkit :

This project was built without heavy frameworks to ensure every line of code was intentional and high-performing:

The Core: Semantic HTML5 and CSS3 (Modern Variables).

The Muscle: GSAP for the cinematic preloader and staggered text reveals.

The Flow: Locomotive Scroll for that "premium" smooth-scrolling feel.

The Smart Layers: Intersection Observer API to make sections "fade and rise" only when you're ready to see them.

## 📖 Presentation Quick-Facts:

If you're looking at this for my competition entry, here is the "under the hood" logic:

Eco-Friendly Performance: I used lazy-loading for all images and force3D: true for animations to keep the site fast and lightweight.

Accessibility First: Every button and link is tagged with aria-labels because the web should be for everyone.

Mobile-First: The site adapts gracefully from a 3-column desktop grid to a single-column mobile stack with a hidden overlay menu.
